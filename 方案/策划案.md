# 《影子穿墙》— 解密游戏策划案 v0.2

> 一个 2.5D 横版解密游戏。**光 → 影子 → 挤过墙缝 → 让墙不再挡人**，整套机制一句话能讲完。
> 场景只有 4 种东西：**玩家、方块、墙、灯**。

---

## 1. 一句话核心

**用光照把影子挤成合适的形状，让影子钻过墙缝去另一边投出方块，墙的两侧都有方块影子时，墙就让你穿过去。**

---

## 2. 场景四元素

| 元素 | 性质 |
| --- | --- |
| **玩家** | 唯一能动的人，2.5D 横版移动，**自己不算开门工具**（玩家自己的影子不参与开门） |
| **方块** | 可移动 / 可堆叠 / 可被光照投射影子。**唯一能开门的"投影源"** |
| **墙** | 默认阻挡玩家；满足条件后**阻挡作用消失**，玩家可穿过 |
| **灯** | 发光体，**不产生影子**（不阻挡别的光） |

> 整个游戏没有敌人、没有机关盒、没有道具栏——纯靠光、方块、墙、灯的组合解谜。

---

## 3. 光与影子

### 3.1 灯 = 玩家手里那束光

- **方向**：360° 自由旋转，控制影子朝哪边投
- **距离**：远 → 影子**大而虚**；近 → 影子**小而实**
- **变形**：通过调整光，影子可以**变高/矮/胖/瘦**（关键能力）

### 3.2 影子的三条性质

1. **平面**：没高度，贴在它落地的平面（地面 / 墙面）上
2. **可变形**：高矮胖瘦随光变（这是穿墙缝的前置能力）
3. **会说话**：作为旁白存在，给玩家台词和提示

### 3.3 影子的核心能力：**钻墙缝**

- 两面墙之间常会有狭窄的缝隙，玩家身体**绝对过不去**
- 影子是平面 + 可变形 → 可以被**挤成长条**钻过去
- 钻过去的影子会**出现在墙的另一边**
- 这是玩家"间接到达墙另一边"的关键路径

### 3.4 影子方块

- 玩家让影子**停下**（操作：放下 / 落定）
- 影子当前位置 → **生成一个实体方块**
- 这个方块从此是真实物体，可以**自己投出自己的影子**
- 远光 = 一次生成多个方块（铺路），近光 = 单个精确生成（拼图）

---

## 4. 开门机制：**墙的两侧都有方块影子 → 墙不再挡人**

### 4.1 默认状态
- 墙是**实墙**，玩家撞上去会被挡住

### 4.2 开门条件
- 墙的**两面**（包括墙缝）都**被某个方块的影子覆盖到**
- 满足条件 → 墙的**阻挡作用消失**，玩家可自由穿过

### 4.3 玩家和灯都不算
- 玩家**自己**的影子**不**算（玩家是"人"，不是"工具"）
- 灯**不**产生影子（灯是发光体）
- **唯一**能开门的"投影源"是**方块**（含影子方块）

### 4.4 一次开门，永久消失
- 墙一旦开门，**整张地图的同构墙**也都不再挡人（避免同一条谜题反复解）
- 死亡 / 重置后开门状态保留

### 4.5 视觉表现
- **未开门**：墙是普通实墙
- **单侧有影子**：墙面出现**半透明门洞预览**（提示玩家"去另一边补一个方块影子"）
- **双侧都有**（两段式开门动画）：
  1. 门位置**中心开始渐变透明**（外圈仍是实体）
  2. 透明区域**向外扩散**，最终**消失一块门大小的墙**
  3. 配合：提示音 + 暂停 0.3s

---

## 5. 墙后危险区（限时挑战）

> 这是整个游戏的**风险点**——也是"关卡联动"的来源。

### 5.1 不是所有墙都安全
- 大部分墙打开后是**普通可走区域**
- 但有一类特殊墙（带标识 / 异色 / 出现在剧情节点）：打开后是**危险区**

### 5.2 危险区规则
- 玩家进入后**开始倒计时**（默认 45s）
- 超时 = 死亡（**镜头 FOV 收缩 + 黑边吞噬** + 强制传送回主区域）
- 死一次 = 计时重置，**已完成的方块开门状态保留**（鼓励多次尝试）

### 5.3 出口
- 危险区里**总有一个发光的门**（通常是让玩家原路返回的墙）
- 出口一直亮着 → 给玩家"退路"的视觉锚点
- 玩家走回出口 → 倒计时取消，安全

### 5.4 一面墙 = 两个关卡（联动）
- 同一堵危险墙的两侧是两个独立谜题
- 主侧谜题可能要求摆方块投出影子
- 墙后侧谜题可能要求在限时内解开机关
- **必须两侧都解开**，这面墙才算"通过"

---

## 6. 完整玩法循环

```
1. 观察谜题（看到墙 + 看到墙缝）
2. 用光照 + 变形 → 让影子挤过墙缝
3. 影子在墙后 → 放下 → 生成方块
4. 方块在墙后被光照 → 投出影子到墙面
5. 主侧玩家也摆方块 / 挪方块 → 主侧墙面也有影子
6. 双侧影子齐 → 墙不再挡人
7. 玩家穿墙 → 进入墙后区域
8. （如为危险区）限时内解谜 → 找发光出口返回
9. 通过 = 进入下一组谜题
```

---

## 7. 谜题设计模式

| 模式 | 描述 | 难度 |
| --- | --- | --- |
| **基本开门** | 摆方块投出双侧影子 | 教程 |
| **影缝接力** | 影子挤过一道缝 → 到第二道缝 → 投到目标墙 | 入门 |
| **变形成败** | 缝太窄 → 必须先变瘦；缝太高 → 必须先变矮 | 入门 |
| **方块反弹** | 影子没法直接投到目标墙 → 用方块当"反光板"改变投射方向 | 进阶 |
| **危险区限时** | 限时内摆方块开门 → 走回出口 | 中等 |
| **墙后控方** | 墙后摆方块 → 投影影响主侧的机关 / 谜题 | 进阶 |
| **多缝迷宫** | 一次要走 3-4 道墙缝，影子变形多次 | 高难 |
| **方块滑落** | 影子投在斜面 / 移动物体上，方块位置实时变化 | 高难 |

## 8. 视觉与镜头

- **画面**：2.5D 侧视角（参考 Limbo / Inside）
- **美术风格**：参考《笼中窥梦》+《超越限空间》 → **极简、超现实、单色调（灰白为主 + 选色高亮）、几何感强**
- **光**：唯一有"亮度"的元素，其他都是黑 / 灰 / 单色
- **影子变形**：玩家调光时，影子有 0.3s 平滑过渡动画
- **开门效果（两段式）**：
  1. 门位置**中心先渐变透明**（中间看穿墙后，外圈仍是实体）
  2. 透明区域**向外扩散**，最终**消失一块门大小的墙**
  3. 配合：提示音 + 暂停 0.3s
- **危险区死亡**：FOV 60° → 25°（2s 内），黑边吞噬，玩家被"吐"回主区域
- **发光出口**：危险区里那个门**永远亮着**，跟其他门视觉差异明显

---

## 9. 关卡结构

- **教程关 1-3**：基本光源控制 → 影子变形 → 钻墙缝 → 摆方块开门
- **基础关 4-8**：单墙开门 + 简单危险区
- **联动关 9-15**：一面墙两个关卡，两侧谜题互相依赖
- **挑战关 16+**：多缝 + 多墙 + 复杂方块机关

> 单关节奏 ≈ 5-8 分钟，**总时长 ≈ 10 小时**（约 60-80 个谜题组）

## 10. 已锁定的设计决策

| 决策 | 结论 |
| --- | --- |
| 场景元素 | 只有 4 种：玩家、方块、墙、灯 |
| 灯是否投影 | 不投影 |
| 玩家影子是否算开门 | 不算 |
| 开门工具 | 只能是方块 |
| 开门条件 | 墙两侧都有方块影子 |
| 开门效果（两段式） | ① 门位置**中心先渐变透明** ② 透明区域**向外扩散**，最终**消失一块门大小的墙** |
| 墙后区域命名 | 不叫"小世界"，就是"墙后区域" |
| 影子是否可穿墙 | 影子可挤过墙缝，玩家可穿已开门的墙 |
| 限时 | 特定墙后区域才限时 |
| 总时长 | **10 小时**（约 60-80 个谜题组） |
| 美术参考 | 《笼中窥梦》+《超越限空间》→ 极简 / 超现实 / 单色调 / 几何感 |
| 故事 | **临时占位版**（已写入 §11）|

## 11. 故事设定（⚠️ 临时占位版，待替换）

> 以下是**为了推进设计**先占位的故事背景。**不是最终版**，后续你想换就换。

- **玩家**：一个**失忆的人**，不知道自己是谁、从哪里来
- **灯**：玩家身上唯一**还没丢失**的东西，是**某个人留给他**的（"那个人"是谁？玩家自己也记不清）
- **影子**：玩家**失忆前的自己**——还记得一切，但只能用声音陪玩家走过来
- **旁白口吻**：**毒舌但温柔**。会吐槽玩家"你又卡了"、会说"还记得你以前多狼戻吗"、会在危险区超时前嘶吼"快走"、会在通关时沉默很久才说"……谢谢你看到了这里"
- **叙事结构**：每开一道门 / 每进一个墙后危险区 = 找回**一段记忆**。通关时玩家拼不出"我是谁"的完整答案，但拼出了**关于那个人的线索**
- **结局走向**（占位）：玩家始终没有完全恢复记忆，但手中灯的光变得**和初始不一样**——影子的旁白不再出现，灯也不会再问玩家问题。玩家走出最后一堵墙时，发现门后站着的人就是**影子声音的主人**

> **这个故事不锁**。任何时候你告诉我"换个方向"，我都重写。
