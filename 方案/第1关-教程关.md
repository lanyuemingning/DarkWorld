# 《影子穿墙》第 1 关 · 教程关 ·「醒来」

> 玩家第一次接触所有核心机制。**不教术语，只教感觉**。
> 预估时长：**6-8 分钟**。
> 故事背景：使用策划案 §11 临时占位版（失忆玩家 + 灯 + 影子旁白）。

---

## 一、整体节奏

| 阶段 | 时长 | 目的 |
| --- | --- | --- |
| 0. 开场黑屏 + 第一句话 | 30s | 建立情绪：孤独、迷茫 |
| 1. 第一束光（光源控制） | 60s | 教"举灯就有光" |
| 2. 影子的第一次变形 | 60s | 教"光能变影子形状" |
| 3. 钻墙缝 | 90s | 教"影子能去我去不了的地方" |
| 4. 放下方块 | 60s | 教"影子能变实体" |
| 5. 双侧影子开门 | 120s | 教**核心机制**："双侧影子齐 → 墙消失" |
| 6. 第一次穿墙 | 30s | 玩家第一次体验"墙不再挡我" |
| 7. 危险区第一课（限时 + 出口） | 90s | 教"墙后有时限，找发光门出去" |
| 8. 收尾（第一段记忆） | 30s | 旁白出现 + 玩家拾取一个物件 |

## 三、剧本（按时间线）

### 0. 开场黑屏（0:00 - 0:30）

- **画面**：全黑，唯一可见是玩家手中的灯（微弱摇曳）
- **声音**：远处的风、近处的呼吸声
- **影子旁白**（从灯后飘出的低沉声）：
  > "……醒醒。"
  > "你睡了多久了？"
  > "……算了吧，你肯定不记得。"
- **玩家操作**：无。黑屏渐亮，灯的暖光映出玩家的轮廓。

### 1. 第一束光（0:30 - 1:30）

- **画面**：玩家站在一个**纯白空房间**中央，灯握在手里。屏幕右下角显示一个**简洁的光标**（提示灯可调整方向）。
- **玩法**：
  - 玩家移动鼠标 / 右摇杆 → 灯转向，**光的方向改变**
  - 玩家滚轮 / 扳机 → 灯远近变化
- **关键反馈**：
  - 第一次举灯，墙上出现玩家自己的影子
  - **影子不发出声音**（此时还没到旁白登场时间）
- **影子旁白**（轻微冷笑）：
  > "你终于肯看了。"

### 2. 影子的第一次变形（1:30 - 2:30）

- **画面**：玩家身前出现一堵**矮墙**（只到玩家胸口高度），矮墙和主墙之间有一条**窄缝**。
- **玩法**：
  - 玩家尝试把灯调近 → 影子**变矮变胖**（贴近地面）
  - 玩家尝试把灯调远 → 影子**变高变瘦**（覆盖更大范围）
- **关键反馈**：
  - 变胖的影子**挤不过**矮墙
  - 变瘦的影子**能伸到**矮墙后面
- **影子旁白**（从光里第一次清晰出声，像突然在旁边）：
  > "……你以前比这聪明。"
  > "试试把那玩意儿挤扁了。"

### 3. 钻墙缝（2:30 - 4:00）

- **画面**：玩家看到矮墙和主墙之间那条**缝隙**。缝隙宽度刚好够影子通过，玩家身体**绝对过不去**。
- **玩法**：
  - 玩家调灯 → 影子**挤成细长条** → 钻过缝隙
  - 影子成功穿过，**出现在墙后**
- **关键反馈**：
  - 玩家走到矮墙前 → 撞墙，无法直接通过
  - 但玩家能看到**自己的影子贴在墙后的地面上**（远光下能看见）
- **影子旁白**（带点惊讶）：
  > "……我过去了。你没过来。"
  > "有意思。"

### 4. 放下方块（4:00 - 5:00）

- **画面**：影子现在在墙后，贴在某个位置。屏幕出现**「放下」**提示（空格 / 鼠标右键）。
- **玩法**：
  - 玩家按「放下」 → 影子当前位置生成一个**实体方块**
  - 方块"啪"地一声落在地上，发出轻响
- **关键反馈**：
  - 方块是**真实物体**，玩家可以走过去（但现在还在墙这边，碰不到）
  - 方块现在在墙后**自己也有影子**（如果墙后有灯照它）
- **影子旁白**（温柔了一点）：
  > "看，那个东西你也可以摸到了。"
  > "只不过……还隔着墙。"

### 5. 双侧影子开门（5:00 - 7:00）⚡ **核心教学**

- **画面**：墙的两侧都需要方块影子。墙后有一个**灯柱**（墙后危险区入口处的灯），照亮墙后方块，方块在墙上投出影子。
- **问题**：现在**只有墙后**有方块影子。**主侧**这面墙上还是空的。
- **玩法引导**（影子旁白**第一次主动给提示**）：
  > "你这一面，墙是干净的。"
  > "再放一个？"
- **玩家操作**：
  - 玩家把灯转向**主侧地面** → 在主侧某个位置投出方块 → 调整方块位置
  - 主侧方块被玩家自己的灯照到 → 在墙上投出影子
  - **墙的两侧**现在都有方块影子了
- **关键反馈**：
  - 墙**正中心先出现一个透明圆圈**（半透明门洞预览）
  - 透明圆圈**向外扩散**
  - 最终**消失一块门大小的墙**（玩家可以直接走过去）
- **影子旁白**（第一次带情绪）：
  > "……开了。"
  > "你第一次把墙推开。"

### 6. 第一次穿墙（7:00 - 7:30）

- **画面**：墙消失后，玩家**第一次走进墙后区域**。**镜头不切**，直接走。
- **关键反馈**：
  - 玩家走过的瞬间，屏幕**轻微震一下**（不是震动，是亮度变化）
  - 墙后比主区域**暗一点**（危险区前置氛围）
- **影子旁白**（终于笑了）：
  > "嘿，墙后是另一个世界。"

### 7. 危险区第一课（7:30 - 9:00）

- **画面**：玩家走进去的瞬间，**屏幕上方出现倒计时**（45s）+ 边缘开始**轻微发暗**。
- **关键提示**：
  - 墙后右上方有一个**发光的圆门**（出口，永远亮）
  - 玩家朝出口走过去时，倒计时**暂停**（出口就在身边 = 安全）
- **玩法**：
  - 玩家在危险区**走两步就找到出口**（这一关的限时就是走过场）
  - 玩家走回出口 → 倒计时清零 → 屏幕恢复正常
- **影子旁白**（毒舌模式上线）：
  > "别在那儿发呆，发光的门看见了没？"
  > "……你不会连这也看不懂吧？"

### 8. 收尾（9:00 - 9:30）

- **画面**：玩家回到主区域。墙后某处地面上，**留下了一个发光的物件**（像是一张旧照片 / 一封信 / 一颗纽扣——具体看后续美术决定）。
- **关键反馈**：
  - 物件自动飘向玩家，玩家按下"拾取"即可
  - 拾取后，屏幕**闪回一帧模糊的画面**（不解释，让玩家好奇）
- **影子旁白**（突然安静，然后很低）：
  > "……"
  > "……这是你的？"
  > "我不记得了。"

## 四、本关教给玩家的核心能力

| 能力 | 教学方式 | 不直接说的术语 |
| --- | --- | --- |
| 控制光源 | 让玩家自由调 | 没说"光源"两字 |
| 影子可变形 | 矮墙 + 窄缝 | 没说"变形"两字 |
| 影子可钻墙缝 | 直接撞墙 | 没说"墙缝"两字 |
| 影子可落为方块 | 玩家按一个键 | 没说"影子方块"两字 |
| 双侧影子开门 | 一次完整开门循环 | 没说"开门机制"两字 |
| 危险区限时 + 出口 | 走过场 | 没说"危险区"两字 |

> 教学哲学：**让玩家自己悟出来**，旁白只点不破。

---

## 五、本关**不教**的机制（留给后续关卡）

- 多墙缝接力
- 影子反复变形（多次挤过不同形状的缝）
- 方块反射光 / 改变投射方向
- 危险区**真的**有难度（这次只是走过场）
- 墙后摆方块影响主侧机关
- 多面墙同时开门

---

## 六、剧情伏笔（这一关埋的线）

1. **影子旁白说"你以前"** → 暗示影子认识玩家的过去
2. **拾取的物件是模糊的** → 玩家不知道那是什么，但影子说"我不记得了"（**不记得 ≠ 不知道**，埋悬念）
3. **灯是别人给的** → 玩家还没意识到，但灯的光在玩家**拿起它**的瞬间闪了一下（暗示灯有生命/记忆）

---

## 七、本关死亡 / 重置规则

- 危险区**不会死**（45s 就是走过场，玩家 10s 内一定找到出口）
- 玩家**可以**在任意时刻手动"重置本关"（按 R 键）→ 重新开始
- 重置后**已拾取的物件**保留（不影响剧情推进）

---

## 八、下一关预告（不显示在游戏里）

第 2 关「第一道真正的墙」：
- 玩家面对一堵**更高的墙**，墙缝**更窄**
- 必须先调光把影子**完全压扁**才能挤过去
- 墙后是第一个**真的**危险区：30s 内摆 3 个方块开门
- 旁白第一次**不提示** → 玩家必须自己记住机制
